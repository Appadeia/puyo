# puyo
```
puyo in, install <packagename>             - Installs a package. You can use both main and AUR packages.
puyo remove, rm <packagename>              - Removes a package.
puyo dup, upgrade, up, update              - Updates your system.
puyo package, pkg <file>                   - Installs a package from a pkg.tar.xz file.
puyo install-aur, inaur <packagename>      - Installs a package from the AUR. Use this if install doesn't work for an AUR package.
puyo install-puylet, inpuylet <puylet>     - Installs a puylet.
puyo remove-puylet, rmpuylet  <puylet>     - Removes a puylet.
```
